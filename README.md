# Safe Space — Where Little Ones Belong

Website for Safe Space, a family childcare learning home in Talking Rock, Georgia.

## Structure

```
index.html                          → Main site (single-page scroll)
CNAME                               → Custom domain config
assets/
  css/style.css                     → Brand design system
  js/main.js                        → Scroll effects & animations
  images/
    logo/                           → Logo files (Fotor)
    illustrations/                  → Nana's Fotor artwork
    photos/                         → Real photos
```

## Adding Artwork

Upload Fotor illustrations to `assets/images/illustrations/` and photos to `assets/images/photos/`.

## Hosting

GitHub Pages from `main` branch, root. Custom domain: safespace.kids
